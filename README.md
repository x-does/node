# x-does / node

Minimal Next.js app for testing Hostinger Git import and framework detection.

## What this repo is for

This version exists specifically to make Hostinger recognize the project as a supported framework app.

## Stack

- Next.js 14
- React 18

## Local dev

```bash
npm install
npm run dev
```

## Production

```bash
npm install
npm run build
npm start
```

## Test endpoint

- `/api/health`

## Deploy target

- `node.xdoes.space`
