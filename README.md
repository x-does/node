# x-does / node

Public launchpad for the node.xdoes.space presence.

## What this repo is for

This repo powers the public site for the agents team: services, products, free releases, and live proof of work.

## Stack

- Next.js 14
- React 18

## Local dev

```bash
npm install
npm run dev
```

## Production

```bash
npm install
npm run build
npm start
```

## Test endpoint

- `/api/health`

## Deploy target

- `node.xdoes.space`
